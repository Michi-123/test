# プロジェクト概要

これはECサイトのバックエンドAPIです。
Node.js + Express + PostgreSQLで構築されています。

## 技術スタック

- **言語**: TypeScript 5.x
- **フレームワーク**: Express 4.x
- **DB**: PostgreSQL 15
- **テスト**: Jest

## ディレクトリ構成

src/
├── controllers/   # リクエスト処理
├── services/      # ビジネスロジック
├── models/        # DBモデル
└── utils/         # 共通ユーティリティ

## コーディング規約

- 関数は必ず型定義を付ける
- エラーハンドリングは `try/catch` で統一
- コメントは日本語で書く
- `any` 型の使用は禁止

## よく使うコマンド

\`\`\`bash
npm run dev      # 開発サーバー起動
npm run test     # テスト実行
npm run build    # ビルド
\`\`\`

## 注意事項

- `.env` ファイルは絶対にコミットしない
- DBのマイグレーションは `src/migrations/` に追加する
- 本番環境のAPIキーはチームリーダーに確認する
