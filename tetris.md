# TETRIS プロジェクト仕様書

## 概要

ブラウザで動作するシンプルなテトリスゲームです。
HTML5 Canvas + Vanilla JavaScript で実装します。

## 技術スタック

- **言語**: HTML5 / CSS3 / JavaScript (ES6+)
- **描画**: Canvas API
- **外部ライブラリ**: なし（純粋なVanilla JS）

## ファイル構成

```
tetris/
├── index.html       # メインHTML
├── style.css        # スタイル
├── game.js          # ゲームロジック
├── board.js         # ボード管理
├── tetromino.js     # テトリミノ定義・操作
└── renderer.js      # 描画処理
```

## ゲーム仕様

### フィールド
- サイズ: 10列 × 20行
- セルサイズ: 32px × 32px
- 背景色: #1a1a2e（ダークネイビー）
- グリッド線: 半透明の白

### テトリミノの種類と色

| ピース | 形状 | 色       |
|--------|------|----------|
| I      | ████ | #00f0f0（シアン） |
| O      | ██   | #f0f000（イエロー） |
|        | ██   |          |
| T      | ███  | #a000f0（パープル） |
|        |  █   |          |
| S      |  ██  | #00f000（グリーン） |
|        | ██   |          |
| Z      | ██   | #f00000（レッド） |
|        |  ██  |          |
| J      | █    | #0000f0（ブルー） |
|        | ███  |          |
| L      |   █  | #f0a000（オレンジ） |
|        | ███  |          |

### 操作方法

| キー          | 動作             |
|---------------|------------------|
| ← →           | 左右移動         |
| ↓             | ソフトドロップ   |
| Space         | ハードドロップ   |
| Z / ↑         | 左回転           |
| X             | 右回転           |
| C             | ホールド         |
| P             | ポーズ           |

### スコアシステム

| アクション         | 得点                        |
|--------------------|-----------------------------|
| 1ライン消去        | 100 × レベル                |
| 2ライン同時消去    | 300 × レベル                |
| 3ライン同時消去    | 500 × レベル                |
| 4ライン（テトリス）| 800 × レベル                |
| ソフトドロップ     | 落下マス数 × 1              |
| ハードドロップ     | 落下マス数 × 2              |

### レベルシステム

- 10ライン消去ごとにレベルアップ
- レベルが上がるほど落下速度が増加
- 落下間隔(ms) = `max(100, 1000 - (level - 1) * 90)`

## UI要素

### メイン画面
- ゲームボード（中央）
- NEXT（次のピース、3つ先まで表示）
- HOLD（ホールドピース）
- SCORE（スコア）
- LEVEL（レベル）
- LINES（消去ライン数）

### 画面状態
- `START`：スタート画面
- `PLAYING`：プレイ中
- `PAUSED`：ポーズ中
- `GAMEOVER`：ゲームオーバー

## コーディング規約

- クラスベースで実装（Game, Board, Tetromino, Renderer）
- マジックナンバーは `constants.js` に定義
- `requestAnimationFrame` でゲームループを管理
- 状態管理は `Game` クラスに集約する

## よく使うコマンド

```bash
# ローカルサーバー起動（Python）
python -m http.server 8080

# ローカルサーバー起動（Node.js）
npx serve .
```

## 注意事項

- テトリミノの回転は **SRS（Super Rotation System）** を採用する
- ゴーストピース（落下位置の半透明表示）を実装する
- モバイル対応のためタッチイベントも追加する
- ゲームオーバー時にハイスコアをlocalStorageに保存する
